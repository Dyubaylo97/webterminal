Caché Web Terminal try v4 beta release!

Showcase

Use full Caché syntax and all additional commands supported by Caché Web Terminal. Change namespace, set variables - feel free as in common terminal! In addition, Caché Web Terminal provides is a lot of new features and options, and you can see some of them on the right-hand side.

Caché Web Terminal is compatible with all mobile devices - control your Caché easily from your pocket!

Enter SQL mode and execute any SQL query right from the terminal.

Intelligent autocomplete feature will help you to complete you input faster. It differs methods from properties, classes from keywords. It suggests variable names, as well as variable properties and methods, if the variable is an object.

Tracing feature enables you to track variables and files, watch for any changes there.

Check other cool features by installing Caché Web Terminal!

How to Install

  1. Download and import latest XML file into any namespace, compiling all the classes. (As system user)
  2. Open http://[host]:[port]/terminal/ (slash at the end is required) web page. (For example, http://localhost:57772/terminal/)
  3. Enter user credentials (_SYSTEM / SYS by default).
  4. That's all!

Notes

  • Your Caché version at least 2013.1 (WebSockets support);
  • User have system privileges to install the terminal;
  • Note that anyone who is able to authorize on /terminal/ web application will have almost full control under your Caché!

Importing WebTerminal.Installer class will also create /terminal WEB-application. The WEB-application will be protected with a password authorization by default.

Use /update terminal command (from CWT v2.0 only) to automatically update terminal application to the newest version.

Please, take a note that starting from v4.0.0-alpha.53, Caché WebTerminal collects anonymous statistics during the installation and usage process. The information WebTerminal collects includes:

  • Success of the installation/update process, error messages occurring during the WebTerminal installation;
  • Version of Caché ($zv), WebTerminal's version;
  • Usage identifier telling whether WebTerminal is used for local or remote sessions.

The only goal of statistics collection is to have analytics over WebTerminal usage by users for creating a better product. Any error reports collected in this way will help me, the developer of WebTerminal to improve the application. This data is completely anonymous and will never include any licensing or Caché usage information, so feel free to use WebTerminal anytime, anywhere, for your needs.

Welcome to Caché Web Terminal homepage!

This is first web-based Caché terminal emulator.

We're glad to introduce something new and convenient to Caché. Today it's a completely new terminal utility in your favourite browser connected over WebSocket to your server. Make your database easy-accessible wherever you are and change your viewpoint about classic gray terminal - future incomes!

This utility will help you to administrate your database and, maybe, will replace standard terminal utility for you at all. Our target is to develop more than terminal - universal control utility to ease administrators work and save a couple of time.
