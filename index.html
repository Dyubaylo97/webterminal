Installation

Just import xml file to %SYS namespace.

Importing %WebTerminal.Installer class will also create /terminal WEB-application. (since version 3.0) The WEB-application will be protected with a password authorization by default.

Usage

Open http://[host]:[port]/terminal/ (slash at end is required!).

For Terminal Versions Under 3.0:

Open http://[host]:[port]/csp/sys/webterminal/index.csp.

Ensure that:

  • Your Caché version at least 2013.1 (WebSockets support);
  • You have write permissions to %CACHELIB database (to install or update terminal); How to enable?
  • Anyone who is able to authorize on /terminal/ web application (index.csp page for terminal versions before 3.0) will have full control under your machine!

Welcome to Caché Web Terminal homepage!

This is first web-based Caché terminal emulator.

We're glad to introduce something new and convenient to Caché. Today it's a completely new terminal utility in your favourite browser connected over WebSocket to your server. Make your database easy-accessible wherever you are and change your viewpoint about classic gray terminal - future incomes!

This utility will help you to administrate your database and, maybe, will replace standard terminal utility for you at all. Our target is to develop more than terminal - universal control utility to ease administrators work and save a couple of time. It will feature a lot of new things, which are not included in typical terminal application. Are you a system administrator or just a Caché developer? Ready to change something in work with your favourite database? Try it right now!
